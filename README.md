# Execute e implante seu aplicativo AI Studio

Este repositório contém tudo o que você precisa para executar seu aplicativo localmente.

Visualize seu aplicativo no AI Studio: https://ai.studio/apps/drive/12hzgdDFtn2JlYzNV0kgqr7s5PTQHC6J7

## Executar Localmente

**Pré-requisitos:**  Node.js

1. Instale as dependências:
   `npm install`
2. Configure a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com sua chave de API do Gemini
3. Execute o aplicativo:
   `npm run dev`
